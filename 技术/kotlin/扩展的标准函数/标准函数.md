#### run的用法

- 提供了一个作用域

- 有一个返回值，是作用域最后一行代码执行的结果

- 代码实操

  ```kotlin
  /**
   * 扩展标准函数run
   * 1. 提供了一个作用域
   * 2. 有一个返回值，返回作用域中最后一行代码的结果
   */
  fun runFun() {
      val str = "zhangqi"
  
      var length = str.run {
          println("length = $length")
          length
      }
  
      print("length = $length")
  }
  ```
