#### 画文字

###### 基线

基线就是我们四格线中的第三条线。只要基线的位置确认了，那么文字的位置就确定了。

###### canvas.drawText

```java
/**
 * text:要绘制的文字
 * x：绘制原点的x轴坐标
 * y：绘制原点的y轴坐标  --->基线的位置
 * paint：用作画的画笔
 */
public void drawText(String text, float x, float y, @NonNull Paint paint)
```

###### paint.setTextAlign

- Paint.Align.LEFT
- Paint.Align.CENTER
- Paint.Align.RIGHT
